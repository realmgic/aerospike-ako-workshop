# Trainee Prerequisites

The instructor provides a pre-built **AWS EKS** lab environment. Trainees typically **observe** the demo; a hands-on variant may share kubeconfig.

## What you need

- Web browser or terminal to follow along
- Basic Kubernetes familiarity (pods, namespaces, kubectl)
- Optional: kubectl installed if running commands yourself

## What the instructor provides

- EKS cluster with AKO installed
- Walkthrough guides in [sections/](../sections/)
- Enterprise Aerospike licensing (feature-key) — not shared with trainees

## Setup leads

If you are building the environment, see [instructor/client-prerequisites.md](../instructor/client-prerequisites.md).

## After setup

Use [checklist.md](checklist.md) to verify the cluster is ready.
